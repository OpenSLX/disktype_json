1.3. Usage Examples

These usage examples use Linux device names. See the next section for a guide to device names on different systems.

Analyzing a CD image: disktype debian-30r0-i386-binary-1_NONUS.iso

Analyzing the first IDE hard disk: disktype /dev/hda

Analyzing a CD or DVD: disktype /dev/cdrom

Analyzing a floppy: disktype /dev/fd0
