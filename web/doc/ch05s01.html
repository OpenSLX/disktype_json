5.1. Usage

Install the tool using GNU make as described in section 1.1.

Run disktype with the path to your desired source as command line argument. Using multiple paths as arguments, you may analyze several sources at a time. disktype will return a single JSON object for each source to standard output.

If you call the tool with a single source only, you may add "| json_pp" to the command in order to receive a formated output.

Check the /misc/file-system-sampler/ folder for some exemplary images.
