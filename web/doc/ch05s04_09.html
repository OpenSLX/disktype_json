5.4.12.13. FAT

Wikidata id: Q190167

Instance of File System.

It's seems to be hard to detect a FAT with certainty. There are several hints for the presence of a FAT though. disktype searches for these hints and stores the result in hints_score

Inherited Properties: 

    + volume_name       (File System)
    + volume_size       (File System)
    + block_size        (File System)

Individual Properties: 

    - cluster_size        {#unsigned long long}           (Size of one cluster in bytes. If identical to block_size, there's only one block in each cluster.)
    - hints_score         {0, ... , 5}                    (The higher, the likelier it's actually a FAT.)
