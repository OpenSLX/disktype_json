5.4.18. Partition

Wikidata id: Q255215

Individual Properties: 

    - kind                {apple, amiga, atari, mbr, gpt, sparc, vtoc, bsd} 
    - name                {#char[256]}
    - number              {#int}                                (partitions are enumerated)
    - size                {#unsigned long long}                 (size in bytes)
    - sector_size         {#int}                                (the size in bytes of a single sector)
    - start_sector        {#unsigned long long}                 (the sector on the disk where this partition starts)

                          (The possible type_names depend on the kind of the partition. See section ? for bsd_types, gpt_types, mbr_types, vtoc_types)

    - type_name           {apple: {Apple_Free, Apple_HFS, Apple_partition_map, ...}
                           amiga: {...}
                           atari: {Standard GEMDOS, Big GEMDOS, Extended, Unknown}
                           mbr:   {#mbr_types}
                           gpt:   {#gpt_types}
                           sparc: -
                           vtoc: {#vtoc_types}
                           bsd: {#bsd_types}
                          }
    - bootable            {true, false}
    - disklabel           {true}                    (true if this partition contains the disklabel)
    - unused              {true}
    - GUID                {#char[256]}
    - drive_name          {#char[256]}
    - letter              {char}                    (instead of an enumeration, partitions may also have letters)
